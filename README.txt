Предметная область: авиаперевозки пассажиров (БД "Авиалинии")
Задача предметной области: спроектировать БД, информация которой будет использоваться для определения доходов, приносимых рейсами, даты и времени прибытия самолета в аэропорт назначения, истечения срока эксплуатации самолета и др.
В БД будет храниться информация о:
• аэропортах (Airport): ID аэопорта, название аэропорта, город, часовой пояс
• маршрутах (Route): ID маршрута, аэропорт вылета, аэропорт назначения, цена билета на маршрут (руб.), продолжительность полета (мин.);
• рейсах (Flight): ID рейса, дата и время вылета, рейс отменен или не отменен (Да/Нет);
• пассажирах (Passenger): ID паспорта, Ф.И.О. пассажира, адрес, телефон;
• самолётах (Airplane): бортовой номер, модель (Боинг-747, Ту-134, Ил-62 и др.), дата изготовления, срок эксплуатации (лет), готовность или не готовность к вы лету (Да/Нет, «Да» означает, что самолет находится в исправном состоянии готов к вылету);
• обслуживающий персонал (Staff): личный номер, Ф.И.О. сотрудника, адрес, телефон, налет (часов), должность.

При проектировании БД учитываются следующие моменты:
• по маршруту могут осуществляться несколько рейсов. Рейс осуществляется по одному маршруту;
• пассажир может купить билеты на несколько рейсов. Одним и тем же рейсом могут лететь несколько пассажиров;
• самолет может назначаться на несколько рейсов. На рейс назначается один самолет;
• у самолета один командир корабля. Командир корабля закреплен за одним самолетом.

Основные классы-сущности БД хранятся в папке src/main/java/airlines/bd
Вспомогательные классы - в папке src/main/java/airlines/utilData
